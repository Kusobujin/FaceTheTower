# 🏰 Face the Tower

Face the Tower é um roguelike turn-based onde você controla um mago que invoca criaturas para subir uma torre infinita.

## 🎮 Sobre o jogo

Você fez um pacto e agora sua alma pertence à torre.
Seu objetivo é subir cada andar, enfrentar inimigos e bosses, e talvez encontrar uma forma de descansar de vez.

## ⚔️ Gameplay

* Sistema de combate **turn-based**
* O jogador controla um **mago (suporte)**
* As batalhas são feitas através de **summons (invocações)**
* Se todos os summons morrerem → **Game Over**

## 🔁 Loop do jogo

1. Escolher o mago
2. Escolher o summon
3. Acessar o lobby
4. Preparar build / upgrades
5. Subir a torre
6. Enfrentar inimigos e bosses
7. Repetir

## 👹 Sistema de Boss

* Boss aparece em andares específicos
* Mais forte que inimigos comuns
* UI diferenciada
